Key Features of DJV2

Automatic Updates

DJV2 seamlessly checks for updates to ensure that the application and its updater are always the latest versions. It uses GitHub’s API to detect the latest releases and downloads them if needed, with user-friendly notifications.

Secure User Authentication

Includes an easy-to-use login and signup system with "Remember Me" functionality. DJV2 uses encrypted storage for secure handling of user credentials, protecting user data at every step.

Customizable Visual Themes

With several themes, including Fleury, Dark, and Light, DJV2 allows users to select a style that matches their preferences, dynamically changing the app's color scheme and UI components to fit the chosen theme.

Automated DOSBox Management

If DOSBox isn’t installed, DJV2 will assist with the setup, downloading the latest version and setting paths. It auto-generates configuration files to simplify launching games and customizing game environments.

Drag-and-Drop Game Management

Simply drag and drop game files or folders into DJV2 to add games. The software scans the directories and builds an organized list of DOS games, allowing users to quickly select and play any game in their collection.

Advanced Key Mapping

Provides a full-featured key mapping dialog, allowing users to assign custom controls for DOSBox games. The settings are saved and can be modified, making it easy to set up controls tailored to different games.

Save and Load Game States

DJV2 supports saving and loading game states, allowing users to capture their progress at any time and resume later. This feature is invaluable for long, intricate DOS games where frequent saving is crucial.

In-App Update Notifications

Notifications alert users about updates to DJV2 or DOSBox, guiding them to install these updates efficiently. The notification system is streamlined, ensuring users are aware of any improvements or new features available.

Integrated Help and Troubleshooting

With an in-app help dialog, users have quick access to troubleshooting information, error messages, and solutions. Common issues are listed with detailed explanations, making it easy to find help without leaving the app.

Enhanced Controls and Shortcuts

DJV2 includes keyboard shortcuts for essential actions such as playing, stopping games, opening settings, and selecting folders. Users can operate DJV2 efficiently with just the keyboard.

Fleury Tribute

The Fleury Tribute theme honors Marc-André Fleury’s legacy with a unique design, including custom colors and fun facts about his career. It’s a thoughtful addition for fans of the hockey star.

Robust Error Logging

DJV2’s logging feature captures errors and important events in the application, aiding troubleshooting and ensuring users can resolve issues quickly.

Automatic DOSBox Configuration

Configures DOSBox with optimal settings based on game requirements, including CPU cycles, memory, and display settings, for the best possible performance.
