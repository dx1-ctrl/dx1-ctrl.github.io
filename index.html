Frequently Asked Questions

What games does DJV2 support?
DJV2 is optimized for DOS-based games using DOSBox. It does not support modern games as it emulates classic DOS systems.
How do I update DJV2?
To update DJV2, simply run the built-in updater provided with the installation. This ensures you stay up-to-date without losing any settings or saved data.
Is DJV2 compatible with Windows?
Yes, DJV2 is compatible with Windows systems and optimized for a smooth DOSBox integration. Support for additional platforms may be considered in future releases.
Is DJV2 open-source?
Currently, DJV2 is not open-source. However, we are open to community suggestions and improvements.
