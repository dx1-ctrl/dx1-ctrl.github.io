About DJV2

DJV2 is a solo project built to provide retro gaming enthusiasts an easy way to enjoy classic DOS-based games. Designed with DOSBox integration, DJV2 makes it simple and hassle-free to launch and manage vintage gaming sessions.

Purpose of DJV2

DJV2 was originally created to simplify the DOSBox emulator setup for my mom, making it easier for her to launch nostalgic games without the hassle of complex configurations. Since then, it has evolved into a tool that I hope can help others enjoy retro gaming just as effortlessly.

Frequently Asked Questions

Which platforms are compatible with DJV2?
DJV2 is currently compatible with Windows. Linux and Mac support are under consideration for future releases.
How do I update DJV2?
Run the built-in updater provided with DJV2's installation to apply the latest updates.
What games does DJV2 support?
DJV2 is optimized for DOS-based games via DOSBox. It does not support modern games.
Is DJV2 open-source?
Currently, DJV2 is not open-source, but I value community suggestions.
