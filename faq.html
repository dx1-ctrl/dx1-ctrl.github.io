Frequently Asked Questions

Which platforms are compatible with DJV2?
DJV2 is currently compatible with Windows. Support for Linux and Mac is under consideration for future updates.
How does DJV2 handle updates?
DJV2 uses GitHub’s API to automatically check for the latest versions of both DJV2 and its updater. If a new version is available, users receive a prompt to update, ensuring they always have the latest features and fixes.
What happens if DOSBox is not installed on my system?
If DOSBox is missing, DJV2 will guide you through installing it. The software can download the installer or look for DOSBox in standard installation directories. Once installed, DJV2 auto-configures DOSBox for optimal performance.
Can I customize game controls in DJV2?
Yes, DJV2 includes a comprehensive key mapping feature, allowing you to assign and customize controls for DOS games to your liking. Simply access the key mapping dialog and make adjustments as needed.
Does DJV2 support visual themes?
Absolutely. DJV2 offers several themes, including Default, Dark, Light, and a special Fleury Tribute theme. Each theme customizes the interface colors and elements for a more personalized experience.
What is the "Fleury Tribute" theme?
The Fleury Tribute theme celebrates hockey legend Marc-André Fleury. It features a custom color scheme, achievements, and fun facts about his career, providing a unique visual experience for fans.
Is it possible to save and load game states in DJV2?
Yes, DJV2 allows you to save and load game states, so you can pause and resume your gameplay at any point. This feature is ideal for longer games that require frequent saving.
How secure is DJV2's login system?
DJV2 prioritizes security with encrypted credential storage and secure HTTPS connections for all login and sign-up interactions. User credentials are safely stored with strong encryption, ensuring your data remains private.
Does DJV2 include any in-app help or troubleshooting?
Yes, DJV2 has an in-app help dialog where users can find solutions to common issues and error explanations. Troubleshooting resources cover topics like missing files, update errors, and game compatibility.
Can I use keyboard shortcuts with DJV2?
Yes, DJV2 includes several shortcuts to enhance navigation, such as Ctrl+P to play a game, Ctrl+S to stop a game, Ctrl+O to open the game folder, and Ctrl+, to open settings.
How does DJV2 handle game organization?
Games can be easily added by dragging and dropping files into DJV2. The application scans the directories, organizes games into a list, and allows for quick selection and launching of titles.
What happens if I encounter an error while using DJV2?
DJV2 has robust logging and error handling. Detailed error messages are logged, and common issues are explained in the help section, allowing users to troubleshoot or reach out for support if needed.
How do I change DOSBox settings in DJV2?
DJV2 auto-generates optimal DOSBox configurations, but users can adjust settings like CPU cycles, memory, and display in the settings menu for a tailored gaming experience.
